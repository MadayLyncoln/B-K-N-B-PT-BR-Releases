# B-K-N-B PT-BR Releases

Repositório público de downloads da tradução PT-BR de Banjo-Kazooie: Nuts & Bolts para uso com o projeto reNut.

O código e os arquivos de trabalho da tradução ficam em um repositório privado. Este repositório existe apenas para distribuir pacotes prontos de release.

## Download

Baixe a versão mais recente em:

https://github.com/MadayLyncoln/B-K-N-B-PT-BR-Releases/releases

## Conteúdo do pacote

Cada release publica um arquivo `.zip` com a estrutura pronta para copiar para a pasta do jogo/reNut.

- `renut.toml`: configuração usada pelo reNut para carregar o jogo com a localidade correta da tradução.
- `assets/loctext/spanish/`: arquivos de texto localizados usados como base para a tradução PT-BR.

## Instalação

1. Baixe o `.zip` da release.
2. Extraia o conteúdo na pasta do jogo/reNut.
3. Confirme que `renut.toml` ficou na raiz da pasta.
4. Confirme que os arquivos da tradução ficaram em `assets/loctext/spanish/`.
5. Abra o jogo pelo reNut.
